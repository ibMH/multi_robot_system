Detailed Description

Provides a easy to use implementation of a generic mobile robot controller.

Member Enumeration Documentation

enum Controller::ControllerType

Specifies the different implemented control laws.

Enumerator
pseudo_inverse 

Control law is based on pseudo inveres the input for generate a output (least squares optimisation)

lypanov 

Control law is based on the lyapunov approache. Output is determined from a lyapunov stable function

angle_distance 

Control law based on linear approach in angle and distance respectively
